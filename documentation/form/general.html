表单控件

所有通用的 表单控件，旨在保持一致性。

支持以下表单控件类：

  • .label
  • .input
  • .textarea
  • .select
  • .checkbox
  • .radio
  • .button
  • .help

它们中的每一个都应该包裹在一个 .control 容器中。
当 表单 中有多个控件时，请使用 .field 类作为 容器，以保持间距的一致性。

{% include snippet.html content=example more=true %} {% include anchor.html name="Form field | 表单域" %}

field 包含以下简单结构:

  • 文本标签 label
  • 表单控件 control
  • 一个可选文本 help
{% include snippet.html content=field_example %}

这个容器会保持表单间距 空间一致性.

{% include snippet.html content=fields_example %} {% include anchor.html name="Form control | 表单控件" %}

control是一个 增强表单功能 的多功能容器，因为它与 控件元素具有相同的高度，所以它 只能包含 以下元素：

  • input
  • select
  • button
  • icon
{% include snippet.html content=control_input_example %} {% include snippet.html content=control_select_example %} {% include snippet.html content=control_button_example %} {% include anchor.html name="With icons | 使用图标" %}

您可以在控件上使用 2 种 修饰符：

  • has-icons-left
  • 和/或 has-icons-right

您还需要在 图标 上添加一个修饰符：

  • 如果使用 has-icons-left，需要添加 icon is-left
  • 如果使用 has-icons-right，需要添加 icon is-right

图标容器的大小将由 input 容器定义。

{% include snippet.html content=icons_example clipped=true %}

您也可以在 下拉框 中添加图标。

{% include snippet.html content=select_icons_example %}

如果控件中包含图标，Bulma 将会确保图标保持 居中，不用担心 input 或 图标的大小。

如果您要同时使用 附件控件,请在 field 容器上添加 has-addons 修饰符:

{% include snippet.html content=addons_example %}

您 只能 在 inputs, buttons, and dropdowns 中追加。

追加一个 静态按钮 将会很有用。

{% include snippet.html content=addons_expanded_example horizontal=true more=true %}

{% include snippet.html content=addons_expanded_fullwidth_example %}

按钮列表

如果您只需要一个 按钮 的列表，请尝试使用新的 buttons 类创建 按钮列表 。

{% include anchor.html name="Horizontal form | 水平表单" %}

如果你想要一个 水平的 的表单控件，请在 field 容器上添加 is-horizontal 修饰符：

  • field-label 侧标签
  • field-body input/select/textarea 的容器

您可以在子元素上使用 is-grouped 或 has-addons 。

{% include snippet.html content=horizontal_form_example horizontal=true more=true %}

为了使标签与每种类型和大小的控件 垂直对齐，.field-label 标签带有 4 种尺寸修饰符：

  • .is-small
  • .is-normal 为任何 .input 或 .button
  • .is-medium
  • .is-large
