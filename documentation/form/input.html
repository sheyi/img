你可以在控件上使用 2 种修饰符 :

  • has-icons-left
  • 和/或 has-icons-right

你还需要在 图标 上添加一个修饰符：

  • 当 has-icons-left 使用时，需要添加 icon is-left
  • 当 has-icons-right 使用时，需要添加 icon is-right

图标容器的大小将由 input 大小决定.
