响应性

Bulma 是一个 移动优先 的框架

{% include anchor.html name="Vertical by default | 默认垂直显示" %}

Bulma 中每个元素都是 移动优先 的，并针对 垂直阅读 进行优化, 并且默认为在移动设备上显示:

  • columns 是垂直堆放的
  • level 组件的子元素将垂直堆放
  • nav 导航菜单将会隐藏

例如, 您可以通过添加 is-mobile 修饰符，强制让columns 或 nav 变为 水平 布局。
