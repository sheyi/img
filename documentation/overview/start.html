入门指南

你只要使用 1 个 CSS 文件 来使用 bulma

你有几种方式 开始使用 Bulma:

  1. 使用 npm 来安装 Bulma 包
  2. 使用 cdnjs CDN 来链接 Bulma 样式表
  3. 使用 GitHub repository 来获得最新的开发版本

1

使用 NPM (推荐):

{% highlight bash %}npm install bulma{% endhighlight %}

2

使用 cdnjs CDN
https://cdnjs.com/libraries/bulma

3

从 Github repository 下载
https://github.com/jgthms/bulma/tree/master/css
