面包屑导航

一个简单的 面包屑 组件用来改善您的导航体验。

{% include meta.html since="0.4.3" variables=true colors=false sizes=true %}

面包屑 组件只需要一个 .breadcrumb 容器和一个 ul 列表。

会自动在 li 标签的 ::before 伪元素上添加分隔符。

您可以在 li 标签上使用 is-active 修饰符来显示当前页面，它将禁用链接。

{% include snippet.html content=breadcrumb_example horizontal=true clipped=true %} {% include anchor.html name="Alignment | 对齐" %}

作为替选方案,可以在 .breadcrumb 容器上使用 is-centered 和 is-right 修饰符。

{% include snippet.html content=breadcrumb_centered_example horizontal=true clipped=true %} {% include snippet.html content=breadcrumb_right_example horizontal=true clipped=true %} {% include anchor.html name="Icons | 图标" %}

您可以使用任意 Font Awesome 图标.

{% include snippet.html content=breadcrumb_icons_example horizontal=true clipped=true %} {% include anchor.html name="Alternative separators | 替换分隔符" %}

您可以选择 4 种分隔符: has-arrow-separator has-bullet-separator has-dot-separator and has-succeeds-separator.
