Pagination | 分页

一个响应式、好用、灵活的 分页

{% include meta.html colors=false sizes=true variables=true %}

分页组件由以下几个元素组成：

  • pagination-previous 和 pagination-next 用于导航
  • pagination-list 其中显示页面项目：
    • pagination-link 页码
    • pagination-ellipsis 范围分隔符

所有元素都是可选的，因此您可以根据需要编写分页。

{% include snippet.html content=pagination_example horizontal=true more=true %}

您可以禁用某些链接，如果它们不活动，或更改活动的页码。

{% include snippet.html content=pagination_options_example horizontal=true more=true %}

在 tablet 上默认列表位于左侧，上一页/下一页 按钮位于右侧，但是您可以通过使用 is-centered 和 is-right 来修改这些元素的 顺序。

{% include snippet.html content=pagination_centered_example horizontal=true more=true %} {% include snippet.html content=pagination_right_example horizontal=true more=true %} {% include anchor.html name="Styles | 样式" %} {% include elements/new-tag.html version="0.6.2" %}

您可以通过添加 is-rounded 修饰符来拥有一个圆角的分页项

分页组件有 3 种额外的尺寸。
您只需要在 pagination 组件上添加 is-small, is-medium, 或 is-large 修饰符。
