Tabs

其他风格的简单的响应式水平导航 选项卡

{% include meta.html since="0.4.4" colors=false sizes=true variables=true %}

标签 只需要一个 tabs 容器和 <ul> 列表
默认 标签样式会有一个底部边框

{% include snippet.html content=tabs_example horizontal=true %} {% include snippet.html content=tabs_example horizontal=true more=true %} {% include anchor.html name="Alignment | 对齐" %}

要对齐选项卡列表，请在 .tabs 容器上使用 is-centered 或 is-right 修饰符：

{% include snippet.html content=tabs_centered_example horizontal=true more=true %} {% include snippet.html content=tabs_right_example horizontal=true more=true %} {% include anchor.html name="Icons | 图标" %}

您可以使用任何 Font Awesome 图标.

{% include snippet.html content=tabs_icons_example horizontal=true more=true %} {% include snippet.html content=tabs_icons_example horizontal=true more=true %} {% include anchor.html name="Sizes | 尺寸" %}

您可以选择 3种尺寸: is-small is-medium 和 is-large.

{% include snippet.html content=tabs_small_example horizontal=true more=true %} {% include snippet.html content=tabs_medium_example horizontal=true more=true %} {% include snippet.html content=tabs_large_example horizontal=true more=true %} {% include anchor.html name="Styles | 样式" %}
如果您想要带有 边框 的经典风格, 只需要添加 is-boxed 修饰符.
{% include snippet.html content=tabs_boxed_example horizontal=true more=true %}

如果您想要互相独立的标签（选择某个将取消其他选择），请使用 is-toggle 修饰符。

{% include snippet.html content=tabs_toggle_example horizontal=true more=true %} {% include elements/new-tag.html version="0.6.2" %}

如果您同时使用 is-toggle 和 is-toggle-rounded，则最前和最后的项目将会变为 圆角。

{% include snippet.html content=tabs_toggle_rounded_example horizontal=true more=true %}

如果您希望选项卡占据 整个宽度，可以使用 is-fullwidth 修饰符。

{% include snippet.html content=tabs_fullwidth_example horizontal=true more=true %} {% include anchor.html name="Combining | 组合" %}

您可以 结合 不通的修饰符。 例如，您可以拥有居中的 盒装结构 或 全宽切换 的标签。
