Dropdown

交互式 下拉菜单

{% include meta.html new=true since="0.4.4" colors=false sizes=false variables=true %}

dropdown 组件是下拉按钮和下拉菜单的容器。

  • dropdown 主 容器
    • dropdown-trigger button容器
    • dropdown-menu 可切换的菜单，默认 隐藏
      • dropdown-content 下拉框, 带有白色背景和阴影
        • dropdown-item 下拉菜单种的每个 项目，可以是 a 或 div
        • dropdown-divider 下拉项目的 分割线

{% include anchor.html name="Hoverable or Toggable | 悬浮 或 可切换" %}

dropdown 组件有 2 个额外的修饰符

  • is-hoverable: 当 悬停 下拉触发器时，下拉菜单会显示出来
  • is-active: 下拉菜单会 一直显示

虽然 CSS :hover 可以完美工作，但是 is-active 类可用于想使用 JavaScript 的用户。
