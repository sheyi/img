Panel

一个可组合的 面板，用于控制紧凑的内容

{% include meta.html colors=false sizes=false variables=true %}

panel 有这几种类型的容器：

  • panel-heading 第一个子元素
  • panel-tabs 用于导航
  • panel-block 可以包含其他元素，例如：
    • control
    • input
    • button
    • panel-icon

panel-block 可以是一个锚标签 <a> 或包含一个复选框 <label> 的标签。
