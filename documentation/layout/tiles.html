{% endhighlight %} {% include anchor.html name="Modifiers | 修饰符" %}

tile 元素有 16 种修饰符:

  • 3 种上下文 修饰符
    • is-ancestor
    • is-parent
    • is-child
  • 1 种定向 修饰符
    • is-vertical
  • 12 种水平尺寸 修饰符
    • from is-1
    • to is-12
{% include anchor.html name="How it works: Nesting | 他是如何工作的：嵌套" %}

所有都是 tile 卡片！要创建 title 网格，您只需要 嵌套 tile 元素。

从 最外层 tile 元素开始嵌套 tile 元素。

您可以根据 12列 网格系统来 调整 所有的 tile 元素。
例如，is-4 将会占用水平 1/3 的空间：

{% highlight html %}
{% endhighlight %}

如果你想让 tile 元素 垂直 堆叠，请在父级 tile 上添加 is-vertical 修饰符

{% highlight html %}
{% endhighlight %}

如果你想为 tile 添加 内容，只需:

  • 添加 任何 你想要的 class类，例如 box
  • 在 tile 上添加 is-child 修饰符
  • 在 父级 tile 上添加 is-parent 修饰符

3层嵌套或更多...

你至少要嵌套3 层:

{% highlight markdown %} tile is-ancestor | └───tile is-parent | └───tile is-child {% endhighlight %}
只要你想，可以嵌套更多层！

当然，你可以嵌套更多层级，并将其混合!

{% highlight markdown %} tile is-ancestor | ├───tile is-vertical is-8 | | | ├───tile | | | | | ├───tile is-parent is-vertical | | | ├───tile is-child | | | └───tile is-child | | | | | └───tile is-parent | | └───tile is-child | | | └───tile is-parent | └───tile is-child | └───tile is-parent └───tile is-child {% endhighlight %}
